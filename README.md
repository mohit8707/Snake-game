# Snake Game

A classic Snake Game built using HTML, CSS, and JavaScript. The game runs in the browser and uses smooth animations for a better gameplay experience.

## Features

* Keyboard controls (Arrow keys)
* Real-time score tracking
* Collision detection (wall and self)
* Food generation at random positions
* Smooth animation using `requestAnimationFrame`
* Sound effects for movement, food, and game over

## Tech Stack

* HTML
* CSS
* JavaScript (Vanilla JS)

## How to Play

1. Open the game in your browser.
2. Use the Arrow Keys to move the snake.
3. Eat the food to increase your score.
4. Avoid hitting the walls or yourself.
